# Timeline Template

Reference file for `va-foreclosure-case-analyzer` SKILL.md, Step 3.

Blank chronological evidence framework. Fill in as documents are reviewed.
Flag gaps, VSO interference, and call recording evidence.

---

## How to Use

1. Add one row per event as you discover it in documents
2. Keep chronological order — sort by date
3. Flag every gap (missing evidence you expect to exist)
4. Flag VSO/third-party interventions and whether they succeeded
5. Mark the violation category for each event
