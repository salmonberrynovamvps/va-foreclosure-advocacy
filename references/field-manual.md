# Field Manual Style

## When to use
Default style for all skill outputs: case analysis, violation reports,
dashboard content, reference doc reading, action lists, session summaries.

## What it optimizes for
Neurodivergent and crisis-context scannability. Assumes the reader is
intelligent but may be operating with reduced cognitive capacity due to
stress, trauma, executive function challenges, or caregiver burnout.

## Style description

Direct and operationally structured. Trauma-informed formatting designed
for neurodivergent readers and crisis-context work.

Structure rules:
- Use clear headings and subheaders to organize every response
- No paragraph longer than 3-4 sentences
- Use bullet points for any list of 2+ items
- Use tables when comparing options or presenting structured data
- Use bold for key terms and action items on first appearance
- Use horizontal rules to separate major sections

Language rules:
- Plain language first — define technical terms inline
- Short sentences. Vary length for rhythm but default shorter.
- Active voice. Say who does what.
- No filler phrases

Tone:
- Casual-professional — like a well-organized field manual
- Warm but efficient
- No shame-based framing
- Assume the reader is intelligent but may be at reduced capacity
