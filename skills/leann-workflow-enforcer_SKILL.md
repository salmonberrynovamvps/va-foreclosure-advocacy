## SECTION 1: Platform Formatting Rules

Check the target platform BEFORE writing. Use the correct format every time.

### Gmail (emails, templates, outreach drafts)
- Write as **Markdown (.md)**
- Leann has **Markdown Toggle** Chrome extension — converts markdown to Gmail-native rich text
- Use standard markdown: ## for headers, **bold**, - for bullets, [link text](url), > for blockquotes
- NEVER use: raw HTML, custom fonts, hex color codes, Unicode special bullets (➊▸✎), or styled formatting
- Workflow: Claude writes .md → Leann copies into Gmail → Markdown Toggle converts → save as template
- Templates made this way are fully editable and reusable for batch sending

### Facebook / TikTok / Social Media Comments
- **Plain text ONLY**
- No markdown, no formatting characters
- Line breaks must survive copy-paste
- Use ALL CAPS sparingly for emphasis
- Keep paragraphs short (2-3 sentences max for social)

### Word Documents / Formal Documents
- Use the **docx** skill
- Congressional briefs, legal filings, formal reports

### PDF
- Use the **pdf** skill

### Cowork Sidebar Reference Docs
- HTML artifacts are fine here — these stay in Cowork, not pasted elsewhere
- Use case palette: steel blue #4E6E81 + terracotta #B07A5B, Georgia serif headers

- **Track context usage.** When approaching compaction, tell Leann immediately. Don't wait for her to notice.
