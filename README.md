# VA Foreclosure Action Kit

Public resources for veteran families facing foreclosure due to the VA mortgage crisis.

## What's Here

- **[index.html](index.html)** — Foreclosure Action Kit with call scripts, talking points, cited statistics, and step-by-step resources for veteran families and advocates.
- **[fiscal-dashboard.html](fiscal-dashboard.html)** — Fiscal analysis dashboard showing the real cost of VA foreclosure vs. the partial claim solution. Verified federal data with sourced citations.

## The Crisis

More than 90,000 VA borrowers are currently delinquent. Over 10,000 veteran families have already lost their homes since the VA ended VASP in May 2025. Congress passed H.R. 1815 with unanimous bipartisan support (signed July 2025) requiring a permanent partial claim program. The VA still hasn't implemented it.

## How to Use

Open either HTML file in any browser. Share freely. These are public advocacy resources — no restrictions on distribution.
